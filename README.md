# 🔗 URL Shortener

A production-ready URL shortening service built with FastAPI. Not just a CRUD app — this one has Redis caching, JWT auth, click analytics, background jobs, and is fully containerized with Docker.

Built as a learning project to go deep on backend engineering with Python — routing, database design, caching strategies, auth flows, and deployment. Every feature was a deliberate choice, not just boilerplate.

---

## ✨ Features

- **Shorten URLs** — paste a long URL, get a short one back instantly
- **Smart redirects** — visits the short link, lands on the original page
- **Click tracking** — every redirect is counted, stats available on demand
- **URL expiry** — links expire after 7 days, returns a proper 410 Gone
- **Redis caching** — hot URLs served from memory, zero DB hit on redirect
- **Write-behind cache** — click counts accumulate in Redis and flush to PostgreSQL every 5 minutes via a background scheduler (APScheduler)
- **JWT Authentication** — register, login, own your URLs
- **URL validation** — Pydantic rejects anything that isn't a valid URL before it touches the database
- **Auto-generated API docs** — FastAPI gives you a live Swagger UI at `/docs`, no extra setup
- **Minimal frontend** — clean UI to shorten URLs, check stats, and manage your links
- **Dockerized** — one command to spin up the entire stack

## 🧠 System Design Highlights

**Cache-aside with write-behind for click tracking**
Short URL redirects are served from Redis. On first visit, the URL is fetched from PostgreSQL and cached with a TTL matching the expiry window (7 days) — so stale cache and expired URLs are never a problem. Click counts accumulate in Redis and are flushed to the DB every 5 minutes by a background job, keeping redirects fast without a DB write on every hit.

**Deterministic hashing for deduplication**
Short codes are generated by MD5-hashing the original URL and taking the first 8 characters. Same URL always maps to the same short code — natural deduplication without extra DB lookups.

**Route ordering matters**
`/stats/{short_code}` is registered before `/{short_code}` to prevent FastAPI from swallowing the stats route as a dynamic path parameter. Small detail, real consequence.

## 🚀 Getting Started

### Prerequisites

- Docker and Docker Compose installed
- That's it

### Setup

**1. Clone the repo**
```bash
git clone https://github.com/Sunistha-Agarwal/url-shortener.git
cd url-shortener
```

**2. Create your `.env` file**
```bash
cp .env.example .env
```

Open `.env` and fill in your values:
```
DATABASE_URL=postgresql://postgres:yourpassword@postgres/url_shortener
REDIS_HOST=redis
REDIS_PORT=6379
BASE_URL=http://localhost:8000
SECRET_KEY=your-long-random-secret-key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_DAYS=7
```

**3. Start everything**
```bash
docker compose up --build
```

That's it. Docker pulls PostgreSQL and Redis, installs dependencies, and starts the server.

**4. Open the app**

| What | URL |
|---|---|
| Frontend | http://localhost:8000 |
| API Docs | http://localhost:8000/docs |

## 📡 API Reference

### Auth

| Method | Endpoint | Description |
|---|---|---|
| POST | `/auth/signup` | Register a new user |
| POST | `/auth/login` | Login and get a JWT token |

### URLs

| Method | Endpoint | Auth | Description |
|---|---|---|---|
| POST | `/shorten` | ✅ Required | Shorten a URL |
| GET | `/{short_code}` | ❌ | Redirect to original URL |
| GET | `/stats/{short_code}` | ❌ | Get click stats for a short URL |
| GET | `/my-urls` | ✅ Required | List all your shortened URLs |

### Example — Shorten a URL

```bash
curl -X POST http://localhost:8000/shorten \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/very/long/url"}'
```

Response:
```json
{
  "short_code": "a1b2c3d4",
  "short_url": "http://localhost:8000/a1b2c3d4"
}
```
